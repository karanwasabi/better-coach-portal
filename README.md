# better-coach-portal

Coach-facing web app (Next.js).

## Prerequisites

- **Node.js** 20+ (LTS recommended)
- **pnpm** (`corepack enable` or [pnpm.io](https://pnpm.io/installation))

## Commands

```bash
pnpm install
pnpm dev      # http://localhost:3000
pnpm build
pnpm start
pnpm lint
pnpm format
pnpm format:check
```

## Tech stack

Next.js (App Router), TypeScript, Tailwind CSS, ESLint.
